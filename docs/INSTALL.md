## Requirements
All the codes are tested in the following environment:
-  Linux (tested on Ubuntu 14.04/
-  Python 3.6+
-  PyTorch 1.1 or higher (tested on PyTorch 1.1, 1,3, 1,5
-  CUDA 9.0 or higher (PyTorch 1.3+ needs CUDA 9.2+)
-  spconv
